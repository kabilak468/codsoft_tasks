# CodSoft Python Internship Tasks

This repository contains Python projects completed during the CodSoft Internship Program. The projects are designed to demonstrate problem-solving skills, user interaction, and basic Python programming concepts through menu-driven applications.

## Tasks

### Task 1 - Contact Management System
A Python-based contact management application that allows users to add, view, search, update, and delete contact details using a menu-driven interface.

### Task 2 - To-Do List Application
A console-based To-Do List application that helps users manage daily tasks by adding, viewing, updating, and deleting tasks efficiently.

### Task 3 - Password Generator
A Python Password Generator that creates secure passwords with different complexity levels such as Easy, Medium, and Hard using random character combinations.

## Technologies Used
- Python
- Command-Line Interface (CLI)
- Lists

- Loops
